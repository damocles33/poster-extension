Getting Started

If you see this page, you should see an icon button in status bar that looks like:

You can open a poster request window via one of three methods:

  1. Click on the icon button in the status bar,
  2. Use the Tools->Poster menu item,
  3. Use the Ctrl-Alt-P (Windows/Linux) or Option-Command-P (Mac) keystroke.

Each time you do so will open a new window that you can use for different requests.
